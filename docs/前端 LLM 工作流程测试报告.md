## ⚠️ 发现的问题

### 问题 1: 规则引擎过于严格

**现象**: "订单完整性检查"规则总是触发 `reject`

**原因**: 规则条件可能过于严格，导致标准订单也被拒绝

**建议**: 优化规则条件或添加更多正面规则

### 问题 2: 前端环境变量配置

**需要配置**:
```env
# admin-web/.env
VITE_API_BASE_URL=http://localhost:3001/api
VITE_SOCKET_SERVER=ws://localhost:3001
```
