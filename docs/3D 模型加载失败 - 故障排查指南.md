# 3D 模型加载失败 - 故障排查指南

> **问题**：模型加载失败  
> **更新时间**：2026-03-01

---

## 🔍 可能原因

### 1. 跨域问题（CORS） ❌

**腾讯云 URL 已过期或不允许跨域访问**

**症状**：
- 控制台显示 `Access-Control-Allow-Origin` 错误
- 网络请求状态码 403

**解决方案**：✅ **已切换到本地模型**

---

### 2. 模型路径错误

**症状**：
- 控制台显示 `404 Not Found`
- 模型文件不存在

**检查步骤**：

```bash
# 检查文件是否存在
powershell -Command "Test-Path 'D:\AAAwork\Agent_3DPrint\ai-3d-head-modeler (1)\public\model.glb'"
```

**预期结果**：`True`

---

### 3. Vite 开发服务器未重启

**症状**：
- 修改代码后模型仍然加载失败
- 旧缓存未清除

**解决方案**：

```bash
# 1. 停止当前服务器（Ctrl+C）

# 2. 清理缓存
cd D:\AAAwork\Agent_3DPrint\ai-3d-head-modeler (1)
rm -rf node_modules/.vite

# 3. 重启服务器
npm run dev
```

## 🐛 详细错误日志

### 查看控制台

1. 按 `F12` 打开浏览器开发者工具
2. 切换到 `Console` 标签
3. 查找红色错误信息

**常见错误及解决方案**：

#### 错误 1：`Failed to fetch`

```
原因：文件不存在或路径错误
解决：检查 public/model.glb 是否存在
```

#### 错误 2：`Access-Control-Allow-Origin`

```
原因：跨域限制
解决：使用本地文件（已解决）
```

#### 错误 3：`Unable to load GLTF`

```
原因：GLB 文件损坏或格式不正确
解决：重新复制模型文件
```
